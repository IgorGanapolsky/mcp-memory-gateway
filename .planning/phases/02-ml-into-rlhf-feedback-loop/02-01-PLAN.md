Create scripts/thompson-sampling.js as a CommonJS module (require/module.exports — matches existing codebase pattern). Implement the following:

CONSTANTS:
- HALF_LIFE_DAYS = 7.0
- DECAY_FLOOR = 0.01
- DEFAULT_CATEGORIES = ['code_edit', 'git', 'testing', 'pr_review', 'search', 'architecture', 'security', 'debugging', 'uncategorized']

FUNCTIONS:

1. timeDecayWeight(timestamp):
   - Call parseTimestamp(timestamp) from './feedback-schema' (Phase 1 helper — do NOT reimplement)
   - If null returned: return DECAY_FLOOR
   - ageDays = (Date.now() - d.getTime()) / (1000 * 60 * 60 * 24)
   - Return Math.max(Math.pow(2, -ageDays / HALF_LIFE_DAYS), DECAY_FLOOR)
   - Source: Research doc "Pattern 1: Thompson Sampling Beta-Bernoulli Update"

2. createInitialModel():
   - Returns object with: version:1, created:ISO, updated:ISO, total_entries:0, categories:{}
   - Each DEFAULT_CATEGORY entry: { alpha:1.0, beta:1.0, samples:0, last_updated:null }

3. loadModel(modelPath):
   - If file exists: try JSON.parse; on parse error fall through
   - If file absent or parse error: return createInitialModel()
   - NEVER call createInitialModel() if file loads successfully

4. updateModel(model, { signal, timestamp, categories }):
   - weight = timeDecayWeight(timestamp)
   - isPositive = (signal === 'positive')
   - cats = (categories && categories.length) ? categories : ['uncategorized']
   - For each cat: if not in model.categories, add default entry { alpha:1.0, beta:1.0, samples:0, last_updated:null }
   - isPositive → alpha += weight; else → beta += weight
   - samples += 1; last_updated = timestamp
   - model.total_entries = (model.total_entries || 0) + 1
   - model.updated = new Date().toISOString()
   - Return model (mutated in place, also returned)
   - Anti-pattern: NEVER apply weight=1.0 — always call timeDecayWeight first

5. getReliability(model):
   - For each [cat, params] in model.categories:
     reliability = params.alpha / (params.alpha + params.beta)
   - Return { [cat]: { alpha, beta, reliability, samples } }

6. samplePosteriors(model):
   - For each [cat, params] in model.categories:
     call betaSample(Math.max(params.alpha, 0.01), Math.max(params.beta, 0.01))
   - Returns { [cat]: float }

7. gammaSample(shape) — Marsaglia-Tsang (2000) algorithm:
   - if shape < 1: return gammaSample(1 + shape) * Math.pow(Math.random(), 1/shape)
   - d = shape - 1/3; c = 1/Math.sqrt(9*d)
   - Loop: x = gaussSample(); v = 1 + c*x; while v <= 0 retry
   - v = v*v*v
   - u = Math.random()
   - if (u < 1 - 0.0331*(x*x)*(x*x)) return d*v
   - if (Math.log(u) < 0.5*x*x + d*(1-v+Math.log(v))) return d*v

8. gaussSample() — Box-Muller via rejection:
   - Loop: u = rand*2-1; v = rand*2-1; s = u*u+v*v; while (s>=1 || s===0)
   - return u * Math.sqrt(-2*Math.log(s)/s)

9. betaSample(alpha, beta):
   - x = gammaSample(alpha); y = gammaSample(beta)
   - return x / (x + y)

module.exports: { timeDecayWeight, loadModel, createInitialModel, updateModel, getReliability, samplePosteriors, HALF_LIFE_DAYS, DECAY_FLOOR, DEFAULT_CATEGORIES }

No external npm packages. No fs operations in the math functions (loadModel uses fs.existsSync + fs.readFileSync — already available in Node.js built-ins).
