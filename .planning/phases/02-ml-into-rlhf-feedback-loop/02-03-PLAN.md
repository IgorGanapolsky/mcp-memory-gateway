   input: new Date(Date.now() - 7*24*60*60*1000).toISOString()
   expected: value in (0.48, 0.52)

3. timeDecayWeight — invalid timestamp:
   input: 'not-a-date'
   expected: exactly 0.01 (DECAY_FLOOR)

4. timeDecayWeight — null input:
   input: null
   expected: exactly 0.01

5. createInitialModel — structure:
   expected: has categories object with DEFAULT_CATEGORIES entries, each with alpha=1.0, beta=1.0, samples=0

6. updateModel — positive signal increments alpha:
   input: fresh model, signal='positive', categories=['testing']
   expected: model.categories.testing.alpha > 1.0 (increased by timeDecayWeight)
   expected: model.categories.testing.beta === 1.0 (unchanged)
   expected: model.total_entries === 1

7. updateModel — negative signal increments beta:
   input: fresh model, signal='negative', categories=['testing']
   expected: model.categories.testing.beta > 1.0
   expected: model.categories.testing.alpha === 1.0

8. updateModel — uncategorized fallback:
   input: signal='positive', categories=[] (empty array)
   expected: model.categories.uncategorized.alpha > 1.0

9. updateModel — unknown category auto-created:
   input: signal='positive', categories=['new_category']
   expected: model.categories.new_category.alpha > 1.0

10. getReliability — correct formula:
    input: model where testing.alpha=3.0, testing.beta=1.0
    expected: reliability.testing.reliability === 0.75 (3/(3+1))

11. samplePosteriors — returns values in [0,1]:
    input: model after 5 updates
    expected: each posterior value is a number in [0,1]

12. loadModel — missing file returns initial model:
    input: path to non-existent file
    expected: returned model has total_entries=0

13. loadModel — reads existing file:
    input: write a temp model file, then loadModel
    expected: returned model has total_entries matching written file

14. Decay floor validation:
    input: timestamp 365 days ago
    expected: timeDecayWeight >= 0.01 (floor enforced)
